The Data Science Process
------------------------

First, let's begin with an overview of the typical **data science process**.

### Key Phases of the Data Science Process

There are five key steps to the data science process:
<img width="958" height="325" alt="image" src="https://github.com/user-attachments/assets/e0769d9c-6322-444a-963c-917fc2c964cc" />


1.  **Collect data**. We need to be able to collect reliable data and ensure that we understand its origin, quality, and meaning.
2.  **Prepare data**. The next phase in the data science process is data preparation. This can take up an enormous percentage of total time--estimates range upwards of 80-90%.
3.  **Train a model**. In this phase, we apply appropriate algorithms to our data, resulting in a trained model. Depending on the algorithm, training may take a considerable amount of time. For example, certain applications of neural networks may take over a month to train fully. Generally, however, model training does not take that much time.
4.  **Evaluate the model**. After we have trained a model, we want to ensure that it meets our expectations in terms of quality. Because the real world is always more complicated than our training data set, we want to make sure that the results look reasonable on non-training before pushing a model out to production. This helps us avoid overfitting to the training data.
5.  **Deploy the model**. Having a model is great, but having it available for use is the natural next step. Historically, deploying a model typically meant rewriting it into a "production" development language like C or C++. Today, it is easy to run a microservice in a language like R or Python and handle prediction needs.

## Elements of AI – Lesson II: Related Fields (Summary)

### 1. Artificial Intelligence (AI)

* AI is a **subfield of Computer Science**.
* It focuses on creating systems that can perform tasks requiring human intelligence such as learning, reasoning, perception, and decision-making.

---

## 2. Machine Learning (ML)

* **Machine Learning is a subfield of AI.**
* ML systems improve their performance automatically by learning from data and experience instead of being explicitly programmed.

**Definition:**

> Systems that improve their performance in a given task with more experience or data.

**Examples**

* Spam detection
* Recommendation systems (Netflix, YouTube)
* Fraud detection

---

## 3. Deep Learning (DL)

* **Deep Learning is a subfield of Machine Learning.**
* Uses complex neural networks with many layers ("deep" refers to the depth of the model).
* Became practical because of increased computing power.

**Examples**

* ChatGPT
* Face recognition
* Speech recognition
* Image classification

Relationship:

```
Computer Science
    └── AI
         └── Machine Learning
                └── Deep Learning
```

---

## 4. Data Science

* Data Science is a broad field combining:

  * Statistics
  * Machine Learning
  * Algorithms
  * Databases
  * Web Applications
  * Domain Knowledge (Business, Healthcare, Finance, etc.)

Purpose:

* Extract insights and value from data.

Data Science often uses AI but is not entirely AI.

---

## 5. Robotics

Robotics involves building machines that can sense, think, and act in the real world.

It combines many AI areas:

* Computer Vision
* Speech Recognition
* Natural Language Processing
* Planning
* Reasoning
* Machine Learning

Most modern robots rely heavily on Machine Learning.

Examples:

* Self-driving cars
* Warehouse robots
* Drones
* Industrial robots

---

## 6. What is a Robot?

A robot consists of:

* **Sensors** → Observe the environment
* **Actuators** → Perform actions
* **Program** → Controls behavior

Examples:

* Autonomous cars
* Factory robots
* Cleaning robots

A chatbot is **not** considered a robot because it has no physical sensors or actuators.

# Key Takeaways

* **Computer Science** is the broadest field.
* **AI** is a branch of Computer Science.
* **Machine Learning** is a branch of AI.
* **Deep Learning** is a branch of Machine Learning.
* **Data Science** overlaps with AI, Statistics, and Computer Science.
* **Robotics** combines AI with physical machines that interact with the real world.
